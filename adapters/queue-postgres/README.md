# @pgshift/adapter-queue-postgres

> Internal PostgreSQL adapter for `@pgshift/queue`.

This package is bundled with [`@pgshift/queue`](https://www.npmjs.com/package/@pgshift/queue).
You do not need to install it directly.

```bash
npm install @pgshift/queue
```

---

For documentation, examples, and API reference, see the main package:
**[github.com/pgshift/pgshift](https://github.com/pgshift/pgshift)**
