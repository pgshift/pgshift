# @pgshift/adapter-cron-postgres

> Internal PostgreSQL adapter for `@pgshift/cron`.

This package is bundled with [`@pgshift/cron`](https://www.npmjs.com/package/@pgshift/cron).
You do not need to install it directly.

```bash
npm install @pgshift/cron
```

---

For documentation, examples, and API reference, see the main package:
**[github.com/mkafonso/pgshift](https://github.com/mkafonso/pgshift)**
