# @pgshift/adapter-state-postgres

> Internal PostgreSQL adapter for `@pgshift/state`.

This package is bundled with [`@pgshift/state`](https://www.npmjs.com/package/@pgshift/state).
You do not need to install it directly.

```bash
npm install @pgshift/state
```

---

For documentation, examples, and API reference, see the main package:
**[github.com/pgshift/pgshift](https://github.com/pgshift/pgshift)**
