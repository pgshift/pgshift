# @pgshift/adapter-search-postgres

> Internal PostgreSQL adapter for `@pgshift/search`.

This package is bundled with [`@pgshift/search`](https://www.npmjs.com/package/@pgshift/search).
You do not need to install it directly.

```bash
npm install @pgshift/search
```

---

For documentation, examples, and API reference, see the main package:
**[github.com/mkafonso/pgshift](https://github.com/mkafonso/pgshift)**
