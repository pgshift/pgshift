# @pgshift/adapter-workflow-postgres

> Internal PostgreSQL adapter for `@pgshift/workflow`.

This package is bundled with [`@pgshift/workflow`](https://www.npmjs.com/package/@pgshift/workflow).
You do not need to install it directly.

```bash
npm install @pgshift/workflow
```

---

For documentation, examples, and API reference, see the main package:
**[github.com/mkafonso/pgshift](https://github.com/mkafonso/pgshift)**
