# @pgshift/adapter-vector-postgres

> Internal PostgreSQL adapter for `@pgshift/vector`.

This package is bundled with [`@pgshift/vector`](https://www.npmjs.com/package/@pgshift/vector).
You do not need to install it directly.

```bash
npm install @pgshift/vector
```

---

For documentation, examples, and API reference, see the main package:
**[github.com/mkafonso/pgshift](https://github.com/mkafonso/pgshift)**
