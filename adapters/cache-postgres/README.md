# @pgshift/adapter-cache-postgres

> Internal PostgreSQL adapter for `@pgshift/cache`.

This package is bundled with [`@pgshift/cache`](https://www.npmjs.com/package/@pgshift/cache).
You do not need to install it directly.

```bash
npm install @pgshift/cache
```

---

For documentation, examples, and API reference, see the main package:
**[github.com/mkafonso/pgshift](https://github.com/mkafonso/pgshift)**
